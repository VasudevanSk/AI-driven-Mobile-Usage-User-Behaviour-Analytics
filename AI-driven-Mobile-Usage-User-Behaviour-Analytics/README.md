# AI-Driven Mobile User Behaviour Analytics

This project implements an end-to-end machine learning system to analyze and predict mobile device usage behavior.

## Features
- Data preprocessing
- K-Means clustering
- Logistic Regression classification
- User behavior prediction
- Visualization

## Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib
- VS Code
- Git & GitHub

## How to Run
1. Clone the repository
2. Install requirements
3. Run the notebook or Python script
